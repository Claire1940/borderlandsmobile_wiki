## Conseils de combat pour les nouveaux Chasseurs de l'Arche

Réussir dans Borderlands Mobile nécessite plus qu'un simple doigt rapide sur la gâchette. Vous devez comprendre la mécanique élémentaire de type "pierre-feuille-ciseaux" pour vaincre les boss les plus coriaces comme Shiv ou les diverses Reines Skag.

*   **Incendiaire (Feu)** : Très efficace contre la "chair" (barres de santé rouges).
*   **Électrique (Électricité)** : Essentiel pour briser les "boucliers" (barres de santé bleues).
*   **Corrosif (Acide)** : L'élément de choix pour l'"armure" (barres de santé jaunes).
*   **Slag/Cryo** : Utilisé pour affaiblir ou geler les ennemis, les rendant plus vulnérables aux autres sources de dégâts.

Gardez toujours une variété d'armes élémentaires dans votre inventaire. Utiliser une arme incendiaire contre un ennemi protégé par un bouclier est nettement moins efficace et gaspillera des munitions précieuses.
