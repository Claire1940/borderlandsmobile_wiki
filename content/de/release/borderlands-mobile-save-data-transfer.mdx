## Kompatibilität von Spielständen im Jahr 2026 verstehen

Bevor Sie eine Übertragung versuchen, ist es wichtig zu wissen, welche Plattformen eine direkte Dateimanipulation ermöglichen. Während Borderlands 3 und die Wonderlands-Serie das Shift-System für einige plattformübergreifende Funktionen nutzen, erfordert eine echte manuelle **Borderlands Mobile Spielstand-Übertragung** oft Zugriff auf das interne Dateisystem Ihres Geräts. Handheld-PCs wie das Steam Deck oder der ASUS ROG Ally funktionieren ähnlich wie Desktops und ermöglichen eine einfache Ordnernavigation, während geschlossene Systeme wie die Nintendo Switch Cloud-basierte Workarounds erfordern.

| Plattformtyp | Übertragungsmethode | Schwierigkeitsgrad | Dateizugriff |
| :--- | :--- | :--- | :--- |
| **Handheld-PC (Steam Deck)** | Manuelles Kopieren/Einfügen | Einfach | Voller Zugriff |

> **💡 Tipp:** Erstellen Sie immer eine sekundäre Sicherung Ihres „Saved“-Ordners auf einem externen Laufwerk oder einem Cloud-Dienst, bevor Sie Dateien verschieben. Ein versehentliches Überschreiben kann hunderte Stunden an Gameplay löschen.

## Manuelle Spielstand-Übertragung: Vom PC auf mobile Handhelds

Wenn Sie Ihren Spielstand von einem Desktop-PC auf einen mobilen Handheld-PC verschieben, erfolgt der Prozess größtenteils manuell. Dies ist der zuverlässigste Weg, um sicherzustellen, dass Ihre spezifischen Charakter-Level und Inventargegenstände korrekt übertragen werden, ohne sich auf potenziell instabile Cloud-Synchronisierungen verlassen zu müssen.

### Schritt 1: Quellcode-Dateien finden
Navigieren Sie auf Ihrem primären PC zum lokalen Speicherverzeichnis. Für die meisten Borderlands-Titel, einschließlich Borderlands 3, befindet sich dieses in Ihrem Dokumente-Ordner.

1. Öffnen Sie den **Datei-Explorer**.
2. Navigieren Sie zu `Dokumente\My Games\Borderlands 3\Saved\SaveGames`.
3. Darin sehen Sie einen Ordner, der mit einer langen Folge von alphanumerischen Zeichen benannt ist.
4. Öffnen Sie diesen Ordner, um Ihre `.sav`-Dateien (z. B. `1.sav`, `2.sav`) und Ihre `profile.sav` zu finden.

### Schritt 2: Zielgerät vorbereiten
Um eine **Borderlands Mobile Spielstand-Übertragung** auf ein Gerät wie das Steam Deck einzuleiten, müssen Sie das Spiel auf dem Zielgerät mindestens einmal starten. Dadurch wird die erforderliche Ordnerstruktur erstellt, die Sie später mit Ihren Hauptdaten überschreiben.

### Schritt 3: Daten verschieben
Sobald die Ordner auf Ihrem Mobilgerät erstellt wurden, kopieren Sie den Inhalt aus dem alphanumerischen Ordner Ihres PCs in den entsprechenden Ordner auf dem Mobilgerät. Wenn Sie auf einem Handheld zwischen der Epic Games- und der Steam-Version wechseln, sehen Sie möglicherweise zwei verschiedene alphanumerische Ordner. Verwenden Sie den Zeitstempel „Änderungsdatum“, um festzustellen, welcher Ordner zu welcher Plattform gehört.

| Dateiname | Enthaltene Daten | Wichtigkeit |
| :--- | :--- | :--- |
| **1.sav** | Individueller Charakterfortschritt | Hoch |
| **profile.sav** | Goldene Schlüssel, Wächterrang, Tresor | Kritisch |
| **metadata.json** | Systemkonfiguration | Niedrig |

## Nutzung von Shift für plattformübergreifende Synchronisierung

Für Spieler, die keinen Zugriff auf das Dateisystem ihres Geräts haben, wie z. B. auf mobilen Konsolen, ist die [offizielle Shift-Website](https://shift.gearboxsoftware.com/) das primäre Werkzeug für eine **Borderlands Mobile Spielstand-Übertragung**. Obwohl sie nicht immer eine Schaltfläche zum Verschieben von Spielständen per Klick bietet, verknüpft sie Ihre Lizenzen und Belohnungen über Plattformen hinweg.

1. **Konten verknüpfen:** Stellen Sie sicher, dass Ihre PlayStation-, Xbox-, Steam- und Nintendo-Konten alle mit demselben Shift-Profil verknüpft sind.
2. **Cloud-Upload:** Suchen Sie im Spielmenü Ihrer Konsole nach der Option „Spielstand hochladen“ (häufig in der Borderlands: Legendary Collection zu finden).
3. **Cloud-Download:** Wechseln Sie zu Ihrem Mobilgerät, melden Sie sich mit demselben Shift-Konto an und wählen Sie „Spielstand herunterladen“.

> **⚠️ Warnung:** Die Cloud-Synchronisierung kann manchmal zu Versionskonflikten führen. Stellen Sie sicher, dass sowohl Ihre Quellplattform als auch Ihr Mobilgerät dieselbe Spielversion ausführen (z. B. v1.0.12), bevor Sie eine Übertragung versuchen.

## Fehlerbehebung bei häufigen Übertragungsproblemen

Selbst mit den richtigen Schritten kann es bei einer **Borderlands Mobile Spielstand-Übertragung** gelegentlich zu Fehlern kommen. Die meisten Probleme resultieren aus Dateibenennungskonventionen oder Verzeichnispfaden.

| Problem | Potenzielle Ursache | Lösung |
| :--- | :--- | :--- |
| **Charakter wird nicht angezeigt** | Falscher alphanumerischer Ordner | Überprüfen Sie die Ordner-ID; Steam und Epic verwenden unterschiedliche IDs. |
| **Fehlende Tresor-Gegenstände** | `profile.sav` fehlt | Stellen Sie sicher, dass Sie die Profil-Datei kopiert haben, nicht nur die Charakter-Datei. |
| **Fehler: Beschädigter Spielstand** | Versionskonflikt | Aktualisieren Sie das Spiel auf beiden Geräten auf den neuesten Patch von 2026. |
| **Cloud-Synchronisierung fehlgeschlagen** | Netzwerk-Zeitüberschreitung | Deaktivieren Sie VPNs und stellen Sie eine stabile WLAN-Verbindung sicher. |

## Erweitertes mobiles Datenmanagement

Für Power-User, die Borderlands über mobile Emulation oder spezialisierte Handheld-Betriebssystemumgebungen ausführen, können die Verzeichnispfade erheblich von den Standard-Windows-Pfaden abweichen. Auf Linux-basierten Handhelds (wie dem Steam Deck) sind die Dateien oft in „pfx“-Ordnern (Proton-Präfixe) versteckt.

**Typischer Linux/Steam Deck-Pfad:**
`~/.local/share/Steam/steamapps/compatdata/397540/pfx/drive_c/users/steamuser/Documents/My Games/Borderlands 3/Saved/SaveGames/`

Wenn Sie eine **Borderlands Mobile Spielstand-Übertragung** in diesen Umgebungen durchführen, stellen Sie sicher, dass Sie sich im „Desktop-Modus“ befinden, um auf die versteckten `.local`-Ordner zuzugreifen. Wenn Sie nicht den richtigen Pfad verwenden, erstellt das Spiel einen neuen, leeren Spielstand, anstatt Ihre übertragenen Dateien zu erkennen.

## FAQ

**F: Kann ich meinen Borderlands-Spielstand vom iOS/Android-Streaming auf den PC übertragen?**

A: Wenn Sie einen Cloud-Streaming-Dienst wie GeForce Now oder Xbox Cloud Gaming nutzen, ist Ihr Spielstand normalerweise an die Cloud der Plattform gebunden (Steam Cloud oder Xbox Live). Um diesen auf einen PC zu verschieben, melden Sie sich einfach beim entsprechenden Desktop-Client an, und die Daten werden automatisch synchronisiert. Eine manuelle **Borderlands Mobile Spielstand-Übertragung** ist nur für lokale Dateiverschiebungen erforderlich.

**F: Werden meine Goldenen Schlüssel bei einer Spielstand-Übertragung mit übertragen?**

A: Goldene Schlüssel sind an Ihre `profile.sav`-Datei und Ihr Shift-Konto gebunden. Wenn Sie die `profile.sav`-Datei während einer manuellen Übertragung kopieren, sollten Ihre Schlüssel erscheinen. Es ist jedoch immer sicherer, sicherzustellen, dass Ihr Shift-Konto ordnungsgemäß mit der neuen Plattform verknüpft ist.

**F: Ist es möglich, Spielstände zwischen Borderlands 2 und Borderlands 3 zu verschieben?**

A: Nein. Spielstandsdateien sind spielspezifisch. Sie können keinen Vault-Hunter von Borderlands 2 nach Borderlands 3 übertragen, obwohl Sie möglicherweise Loyalitätsbelohnungen erhalten, wenn Spielstände aus früheren Spielen auf Ihrem Gerät erkannt werden.

**F: Funktioniert die Borderlands Mobile Spielstand-Übertragung für die Nintendo Switch-Version?**

A: Die Switch-Version verwendet primär die im Hauptmenü befindliche „Cloud-Save“-Funktion für die Übertragung zwischen verschiedenen Switch-Konsolen. Das Verschieben eines Switch-Spielstands auf einen PC oder einen anderen mobilen Handheld erfordert Software von Drittanbietern und Hardware-Modifikationen, die offiziell nicht unterstützt werden.
