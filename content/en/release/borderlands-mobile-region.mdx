## Technical Limitations and Settings

As this is a "secret" launch intended for testing, the current build is missing several quality-of-life features that mobile gamers usually expect. The settings menu is currently bare-bones, which can make high-level play difficult for those used to competitive mobile shooters.

*   **No Audio Settings:** You cannot currently adjust music, SFX, or dialogue volumes independently.
*   **No HUD Customization:** The button layout is fixed, meaning "four-finger claw" players will have a hard time adjusting.
*   **Sensitivity Issues:** The default aiming sensitivity is reported to be quite low, making it difficult to track fast-moving Psychos or flying enemies.
*   **Resolution:** On larger devices like iPads, the resolution may appear stretched or pixelated as the game is not yet optimized for all screen sizes.

Despite these hurdles, the gameplay feels "solid" and captures the essence of the franchise. The quick-fire missions are designed for 5-10 minute play sessions, making it perfect for gaming on the go.
