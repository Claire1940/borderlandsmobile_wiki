## Manual Save Transfer: From PC to Mobile Handhelds

If you are moving your save from a desktop PC to a mobile handheld PC, the process is largely manual. This is the most reliable way to ensure your specific character levels and inventory items move correctly without relying on potentially unstable cloud syncs.

### Step 1: Locating the Source Files
On your primary PC, navigate to the local save directory. For most Borderlands titles, including Borderlands 3, this is found within your Documents folder.

1. Open **File Explorer**.
2. Navigate to `Documents\My Games\Borderlands 3\Saved\SaveGames`.
3. Inside, you will see a folder named with a long string of alphanumeric characters.
4. Open this folder to find your `.sav` files (e.g., `1.sav`, `2.sav`) and your `profile.sav`.

### Step 2: Preparing the Target Device
To initiate a **borderlands mobile save data transfer** to a device like the Steam Deck, you must first launch the game on the target device at least once. This creates the necessary folder architecture that you will later overwrite with your main data.

### Step 3: Moving the Data
Once the folders are generated on your mobile device, copy the contents from your PC's alphanumeric folder into the corresponding folder on the mobile device. If you are moving between Epic Games and Steam versions on a handheld, you may see two different alphanumeric folders. Use the "Date Modified" timestamp to identify which folder belongs to which platform.

| File Name | Data Contained | Importance |
| :--- | :--- | :--- |
| **1.sav** | Individual Character Progress | High |
| **profile.sav** | Golden Keys, Guardian Rank, Bank | Critical |
| **metadata.json** | System Configuration | Low |

## Using Shift for Cross-Platform Syncing

For players who cannot access their device's file system, such as those on mobile consoles, the [Official Shift Website](https://shift.gearboxsoftware.com/) is the primary tool for a **borderlands mobile save data transfer**. While it does not always offer a "click to move" save file button, it links your licenses and rewards across platforms.

1. **Link Accounts:** Ensure your PlayStation, Xbox, Steam, and Nintendo accounts are all linked to the same Shift profile.
2. **Cloud Upload:** Within the game menu on your console, look for the "Upload Save" option (common in the Borderlands: Legendary Collection).
3. **Cloud Download:** Switch to your mobile device, log into the same Shift account, and select "Download Save."

> **⚠️ Warning:** Cloud syncing can sometimes result in version mismatches. Ensure both your source platform and your mobile device are running the same game version (e.g., v1.0.12) before attempting a transfer.

## Advanced Mobile Data Management

For power users running Borderlands via mobile emulation or specialized handheld OS environments, the directory paths may differ significantly from standard Windows paths. On Linux-based handhelds (like the Steam Deck), the files are often hidden within "pfx" folders (Proton prefixes).

**Typical Linux/Steam Deck Path:**
`~/.local/share/Steam/steamapps/compatdata/397540/pfx/drive_c/users/steamuser/Documents/My Games/Borderlands 3/Saved/SaveGames/`

When performing a **borderlands mobile save data transfer** in these environments, ensure you are in "Desktop Mode" to access the hidden `.local` folders. Failure to use the correct path will result in the game creating a fresh, empty save instead of recognizing your transferred files.

## FAQ

**Q: Can I transfer my Borderlands save from iOS/Android streaming to PC?**

A: If you are using a cloud streaming service like GeForce Now or Xbox Cloud Gaming, your save is typically tied to the platform's cloud (Steam Cloud or Xbox Live). To move this to a PC, simply log into the corresponding desktop client, and the data will sync automatically. A manual **borderlands mobile save data transfer** is only required for local file moves.

**Q: Will my Golden Keys transfer during a save move?**

A: Golden Keys are tied to your `profile.sav` file and your Shift account. If you copy the `profile.sav` file during a manual transfer, your keys should appear. However, it is always safer to ensure your Shift account is properly linked to the new platform.

**Q: Is it possible to move saves between Borderlands 2 and Borderlands 3?**

A: No. Save files are game-specific. You cannot transfer a Vault Hunter from Borderlands 2 into Borderlands 3, though you may receive loyalty rewards for having save data from previous games detected on your device.

**Q: Does the borderlands mobile save data transfer work for the Nintendo Switch version?**

A: The Switch version primarily uses the in-game "Cloud Save" feature found in the main menu for transferring between different Switch consoles. Moving a Switch save to a PC or another mobile handheld requires third-party software and hardware modifications which are not officially supported.
